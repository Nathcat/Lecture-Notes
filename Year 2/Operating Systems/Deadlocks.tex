\section{Deadlocks}

A system consists of a set $R$ of $m$ resource types. Each resource type $R_i$ has $W_i$ instances.
A process may make use of a resource with:
\begin{itemize}
    \item $request$
    \item $use$
    \item $release$
\end{itemize}
